Json序列化

json与序列化

序列化是指将内存中的对象转换为可存储或传输的字节序列

# 把dict(字典)转换为字符串(序列化)
json.dumps()
# 把字符串转换为dict(字典)(反序列化)
json.loads()
----------------------------------------------------
#load和loads的区别(反序列化)
1.load:针对文件句柄,将json格式的字符串转为dict,从文件中读取
       	json = json.load(open('demo.json','r'))
2.loads:针对内存对象,将字符串转换为dict
    	json = json.loads('{"a":"1","b":"2"}')
        
#dump和dumps(序列化)    
1.dump:将dict转换为json字符串格式并写入到文件
    	dict={"a":"1"}
        json.dump(dict,open('demo.json','w'))
2.dumps:将dict转换为字符串类型
    	dict={"a":"1"}
    	str=json.dumps(dict)

总结

根据序列化和反序列的特性

loads： 是将string转换为dict

dumps： 是将dict转换为string

load： 是将里json格式字符串转化为dict，读取文件

dump： 是将dict类型转换为json格式字符串，存入文件
